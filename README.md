# Quiz With Shuffle (C++)

**Quiz With Shuffle** is a simple console-based **C++ quiz program**.  
It allows users to log in, answer multiple-choice questions, and get their score as a percentage.  
The options for each question are **shuffled randomly** each time, making the quiz more fun and challenging.

---

## Features
- **Login Authentication**: Users must enter a correct password to access the quiz.
- **10 Multiple-Choice Questions**: Each question has 4 options (A, B, C, D).
- **Shuffled Options**: The order of options changes every time the quiz is run.
- **Score Calculation**: At the end, your **percentage score** is displayed.
- **Simple Console Interface**: No extra setup required; works on any system with a C++ compiler.
